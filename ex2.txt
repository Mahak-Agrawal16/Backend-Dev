# Backend Class - Node.js Template

Simple Node.js + Express boilerplate for a backend class.

Structure
- index.js - app entry
- routes/users.js - user routes
- controllers/users.js - simple in-memory controller
- .env.example - example env
- package.json - scripts & deps
- .gitignore

Quick start
1. Copy files into your repo root.
2. Install deps: npm install
3. Start: npm run dev (requires nodemon) or npm start

Endpoints
- GET /api/health -> { status: "ok" }
- GET /api/users -> list users
- POST /api/users -> create user { "name": "..." }

Notes
- This is a learning template with in-memory data. Replace with a DB for production...
